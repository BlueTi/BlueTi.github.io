- 정수 N에 반복적으로 곱해준다.

---


```cpp
#include <iostream>
using namespace std;

int main() {
	int n,sum=1;
	cin>>n;
	for(int i=1;i<=n;i++){
		sum*=i;
	}
	cout<<sum;
	return 0;
}
```
