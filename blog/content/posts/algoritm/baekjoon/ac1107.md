---

- 시작 100이니 목적지에서 100을 절대값을 넣고 찾기 시작한다.
- 한글자씩 떼서 확인하는것이 한칸씩 움직이는것과 비교했을때 뭐가 더 적은가를 비교하며 반복한다.

---

```java

N = int(input())
M = int(input())
enable = {str(x) for x in range(10)}
if (M != 0):
    enable -= set(input().split())

result = abs(N - 100)
for i in range(1000001):
    is_true = True
    for c in str(i):
        if (c not in enable):
            is_true = False
            break
    if is_true is True:
        result = min(result, abs(N - i) + len(str(i)))

print(result)

```
