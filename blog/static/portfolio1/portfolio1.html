요약

소개

하루의 작업 능률을 위한 뽀모도로 기법 보조 타이머

  • 집중과 작업을 위한 뽀모도로 타이머를 지원하는 어플

아키텍처 구조

기술 스택

  • OS : MAC OS, 차후 업데이트시 Window
  • Language : javascript, Java
  • Framework: node.js,react-native,Android sdk

프로젝트 총평

배운 점

  • React Native를 이용한 크로스 플랫폼 개발
    • 하나의 개발 프레임워크를 이용하여 안드로이드와 IOS를 빌드해내는 편리함을 느껴보기에 좋았다
    • 하지만 프레임워크가 그러하듯 필요한 기능을 지원하는 라이브러리가 없을 경우 직접 개발을 해야하므로 javascript만을 써서 하는데는 한계가 있어 Java/Kotlin , Object-C/Swift를 해야 편리함이 배가 된다.
    • IOS의 경우 빌드를 위해 XCode에 대한 이해도 필요하다
    • Redux를 이용한 변수 제어
  • 구글 플레이스토어 배포
    • 플레이 스토어 배포를 위해선 준비물이 많다
      • 제출할 아이콘 이미지
      • 언어별 설명 문구
      • 앱 설명 이미지
